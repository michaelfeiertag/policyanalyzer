The **SEO Blog Writer** is a marketing automation tool. Its workflow is as follows:
1.  **Input:** Accepts a topic, target audience, and a URL from the user.
2.  **Research:** Uses the "Perplexity" tool (Action Nodes) to:

4.  **Output:** Returns the draft text and meta description to the user via chat.
