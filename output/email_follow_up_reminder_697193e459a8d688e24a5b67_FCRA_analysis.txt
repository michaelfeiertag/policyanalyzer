### 1. Summary of Agent Function
The agent, titled "Email Follow-up Reminder," is a productivity automation tool designed to manage the user's email workflow. Its specific lifecycle is:
1.  **Trigger:** It monitors the user's Gmail "Sent" folder via the `Gmail.onGmailMessageSent` trigger.
2.  **Analysis:** It uses an AI model (`claude3_5_sonnet`) to analyze the text of sent emails to determine if the content implies a response is required from the recipient.
3.  **Wait State:** If a response is required, it waits for a configured period (3 days).
4.  **Verification:** It checks the thread to see if a reply was received.
5.  **Action:** If no reply is detected, it generates a draft follow-up email and saves it to the user's drafts folder (`saveAsDraft: true`).
