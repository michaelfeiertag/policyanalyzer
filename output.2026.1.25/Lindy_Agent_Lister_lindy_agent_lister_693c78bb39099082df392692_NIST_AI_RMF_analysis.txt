### 1. Summary of Agent Function
The "Lindy Agent Lister" is an automated utility designed to run on a recurring hourly schedule. Its purpose is to query the Lindy API to retrieve a list of all agents within the current user's workspace. It then iterates through this list to update or upsert rows in a Google Sheet, recording the Agent ID, Agent Name, and Last Task Date for each agent found.

### 2. NIST AI RMF Applicability
**Applicable.**
NIST AI RMF applies to this agent because it creates an automated system that processes organizational data (metadata about other AI agents) and autonomously transmits that data to an external third-party storage provider (Google Sheets). As a "security and compliance officer," the system falls under the **Manage** and **Map** functions of the framework, specifically regarding the **Secure & Resilient** and **Privacy-Enhanced** characteristics of trustworthy AI.
